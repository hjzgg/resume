该产品是推动建筑业进行数字化转型的引擎。项目的整体架构包括1.前台应用：建造管理、现场管理、总承包管理、投资管理、职能管理；2.中台运营：开发中心、流程中心、门户中心、数据中心、设备中心、物料中心、客户中心、供方中心、用户中心；3.后台支撑：同一认证服务、同一权限服务、同一审计服务、同一通信服务、同一流程服务、调动服务、移动中间件、分布式存储、分布式服务框架、分布式缓存、分布式数据库服务、消息队列。 开发技术路线包括：前端展示层（react+redux+webpack）；服务器端负载均衡（nginx）,web缓存（nginx），web服务框架（springmvc+spring data jpa+freemarker）,安全框架（shiro）；服务层搜索服务（solr），消息队列服务（rabbitmq），缓存服务（redis），分布式服务框架（dubbo），文件服务（fastdfs）,调度服务（quartz）；存储层数据库（mysql，oracle），索引数据，文件存储（fastdfs, mongo db）。

负责中台运营中的 门户中心（工作台的集成和维护，普通登录和扫码登录，研究密码加密策略sh1、md5、rsa、des等，提供用户上下文，shiro+cas完成单点登录，解决微服务之间、前后端之间cookie跨域的问题），用户中心（负责用户基本的CRUD,提供web端和移动端的用户认证服务），权限中心（建立角色体系，形成角色-岗位字典-岗位-人员-用户的模型，提供dubbo和rest服务。使用react建立webUI，并提供多个相关的组件，为工作台中的应用提供方便和服务），组织中心（负责人员、部门、组织的CRUD，整合人员1+N方案，提供相关的参照接口和参照的配置，供应用方访问）。
